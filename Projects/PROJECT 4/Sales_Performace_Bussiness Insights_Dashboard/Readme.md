# Sales Performance & Business Insights Dashboard (Power BI)

## Project Overview
This project focuses on analyzing sales performance using Power BI. The objective is to transform raw sales data into a structured and interactive dashboard that provides clear and actionable business insights for decision-making.

---

## Objectives
- Analyze overall sales, profit, and quantity
- Identify sales trends over time
- Compare performance across product categories and regions
- Enable interactive data exploration using slicers

---

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel (Global Superstore Dataset)

---

## Dataset Description
- Dataset: Global Superstore
- Primary Table: Orders
- Key Columns: Order Date, Sales, Profit, Quantity, Category, Region

---

## Data Preparation
- Selected only the required Orders table
- Renamed important columns for clarity and consistency
- Verified and corrected critical data types
- Checked and handled null values in KPI-related columns
- Performed data cleaning using Power Query

---

## DAX Measures
The following DAX measures were created:
- Total Sales
- Total Profit
- Total Quantity

---

## Dashboard Features
- KPI cards displaying Total Sales, Total Profit, and Total Quantity
- Monthly sales trend visualization using a line chart
- Sales comparison by category using a column chart
- Sales comparison by region using a bar chart
- Interactive slicers for Date, Region, and Category

---

## Key Insights
- Sales performance differs significantly across regions
- The Technology category contributes the highest sales
- Monthly sales trends highlight periods of growth and decline
- High sales volumes do not always result in high profit, indicating margin variation

---

## Conclusion
The dashboard provides a clear and business-focused view of sales performance. It enables stakeholders to monitor key metrics, analyze trends, and make informed, data-driven decisions.
